# Roadmap

## Phase 1: Hardware

Build and test the physical server.

## Phase 2: Proxmox

Install Proxmox, configure networking, storage and VM templates.

## Phase 3: TrueNAS

Create a TrueNAS VM, pass through the HBA and build the storage pool.

## Phase 4: Migration

Move data from the old TrueNAS system to the new system.

## Phase 5: Windows Server Lab

Build a Windows Server environment with Active Directory, DNS, DHCP and Group Policy.

## Phase 6: Docker

Deploy Docker services such as Jellyfin, Pi-hole, Grafana, Uptime Kuma and Nginx Proxy Manager.

## Phase 7: Monitoring and Backups

Add monitoring, alerting, UPS integration and backup testing.

## Phase 8: Continuous Improvement

Document fixes, changes, mistakes and future upgrades.
