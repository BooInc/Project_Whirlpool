\# Hardware Installation



\*\*Date:\*\* 12 July 2026



\## Objective



Assemble the Home Server hardware and verify that all components are working correctly before installing an operating system.

| Component | Model |

| Case | Jonsbo N6 |

| Motherboard | ASUS Pro WS W680M-ACE SE |

| RAM | 64GB DDR5 |

\## Tasks Completed



\- Installed motherboard

\- Installed CPU

\- Applied thermal paste

\- Installed CPU cooler

\- Installed memory

\- Installed NVMe SSD

\- Installed power supply

\- Connected case wiring

\- Performed cable management

\- Verified all fans were operating



\---



\## Issues Encountered



\### Front Panel Header



The Jonsbo front panel connector did not match the ASUS workstation motherboard layout.



\*\*Resolution\*\*



Used the ASUS front panel adapter supplied with the motherboard.



\---



\## Validation



\- Successful POST

\- CPU detected correctly

\- Memory detected correctly

\- NVMe SSD detected

\- CPU temperatures normal



\---



\## Lessons Learned



\- Workstation motherboards may use different front panel layouts than consumer boards.

\- Always perform a POST test before completing cable management.

\- Keep all motherboard accessories as they often include proprietary adapters.
