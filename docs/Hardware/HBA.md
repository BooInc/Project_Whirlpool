# HBA

## Planned Model

LSI 9300-16i in IT Mode

## Purpose

The HBA will be passed through directly to the TrueNAS VM. This allows TrueNAS to see and manage the HDDs directly.

## Required Cables

- SFF-8643 to 4 x SATA forward breakout cables

## Important

Forward breakout cables are required. Reverse breakout cables are not suitable for this use.
