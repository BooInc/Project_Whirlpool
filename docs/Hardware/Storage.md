# Storage Plan

## NVMe Storage

Planned:

- 2TB NVMe SSD initially
- Second 2TB NVMe SSD later

Purpose:

- Proxmox OS
- VM disks
- Containers
- ISO files
- Templates

## TrueNAS HDD Pool

Planned:

- 6 x 8TB to 10TB HDDs
- RAIDZ2

Purpose:

- Main storage
- Media
- Documents
- General NAS shares

## Backup HDD

Planned:

- 10TB HDD connected to motherboard SATA

Purpose:

- Proxmox backups
- VM backups
- Important configuration backups

## Offline Backup

A second 10TB drive may be used as offline cold storage for important data.
