# CPU

## Model

Intel Core i7-14700

## Why I chose it

- Strong multi-core performance for virtual machines
- Intel Quick Sync support for Jellyfin/Plex transcoding
- Good long-term performance headroom
- More suitable for this build than a lower-end CPU because the server will run multiple services

## Key feature: Intel Quick Sync

Quick Sync is Intel's built-in video encoding and decoding engine. It allows Jellyfin or Plex to transcode video efficiently without needing a dedicated GPU.
