# Motherboard

## Model

ASUS Pro WS W680M-ACE SE

## Why I chose it

- Micro-ATX form factor
- Intel W680 workstation chipset
- Supports Intel LGA1700 CPUs
- Supports ECC UDIMM memory if upgraded later
- Dual Intel 2.5GbE ports
- Dedicated Realtek 1Gb management port
- PCIe expansion for an LSI HBA
- Suitable for a compact workstation-style server

## Notes

The management port will be connected to the switch separately from the main 2.5GbE network port.

Initial RAM may be non-ECC due to current pricing, but the board allows ECC UDIMM memory later.
