# Power Supply

## Model

SilverStone SX750 Platinum

## Why I chose it

- SFX size is better suited to the compact Jonsbo N6 case
- 750W gives plenty of headroom
- Platinum efficiency
- Fully modular cables

## Expected Load

The planned system should run well below 750W even with HDD spin-up, an i7 CPU, HBA, NVMe drives and multiple HDDs.
