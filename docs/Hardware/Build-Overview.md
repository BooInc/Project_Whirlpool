# Build Overview

## Purpose

Project Whirlpool is being built as a compact home server for storage, virtualisation and IT learning.

## Planned Uses

- Proxmox virtualisation
- TrueNAS storage
- Jellyfin media serving
- Windows Server lab
- Active Directory testing
- Linux administration
- Docker containers
- Backups and monitoring

## Current Build

| Component | Model |
|---|---|
| Case | Jonsbo N6 |
