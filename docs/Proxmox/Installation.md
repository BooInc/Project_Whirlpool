# Proxmox Installation

## Goal

Install Proxmox VE on the NVMe SSD.

## Planned Steps

1. Create Proxmox installer USB.
2. Boot from USB.
3. Install Proxmox to NVMe.
4. Configure management IP.
5. Log into the web interface.
6. Apply updates.
7. Configure repositories.
8. Upload ISO files.

## Notes

Proxmox will be the host OS. TrueNAS will run as a VM.
