# Design Decision: W680 Workstation Platform

## Decision

Use an ASUS Pro WS W680M-ACE SE motherboard.

## Reason

- Workstation-class motherboard
- Micro-ATX
- Supports Intel Quick Sync
- Supports ECC UDIMM if upgraded later
- Good PCIe expansion
- Dedicated management port

## Trade-off

W680 boards are harder to find and more expensive than consumer boards.
