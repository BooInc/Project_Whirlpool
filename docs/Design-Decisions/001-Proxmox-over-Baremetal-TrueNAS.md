# Design Decision: Proxmox over Bare-Metal TrueNAS

## Decision

Use Proxmox as the host OS and run TrueNAS as a VM.

## Reason

This allows the server to be more than just a NAS. It becomes a virtualisation platform for learning Windows Server, Linux, Docker and networking.

## Trade-off

This is more complex than installing TrueNAS directly, but it provides much more flexibility and learning value.
